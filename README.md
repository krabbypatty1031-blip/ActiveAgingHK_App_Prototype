# 🏥 香港老年人智能社区照护App

专为香港长者设计的移动端优先智能社区照护应用，支持"原居安老"理念。

## 🎯 项目特色

### 核心功能
- **健康监测**：交通灯颜色编码显示生命体征（心率、血压、步数、睡眠）
- **紧急求助**：SOS一键求助 + 跌倒检测 + 30秒倒计时取消
- **AI语音助手**：粤语语音识别和合成，支持日常查询和紧急关键词检测
- **家庭联络**：语音描述相册、一键视频通话、社区活动报名
- **家属监控**：Web版仪表盘，实时查看长者健康状态

### 无障碍设计
- **字体大小**：支持16px-28px四种尺寸
- **高对比度**：黑白高对比模式
- **语音导航**：全程粤语语音播报
- **触摸优化**：按钮≥48x48dp，防误触设计
- **简化操作**：最多两层菜单结构

## 🚀 快速开始

### 安装依赖
```bash
npm install
```

### 启动开发服务器
```bash
npm run dev
```

### 访问应用
- 长者端：http://localhost:5173
- 家属端：http://localhost:5173/family

## 🎨 设计规范

### 颜色系统
- 背景：#FFFFFF / #F5F7FA（浅色系）
- 文字：#333333（深色系）
- SOS按钮：#FF4D4F（高饱和红色）
- 状态指示：绿色=正常，黄色=注意，红色=异常

### 字体规范
- 最小字号：18pt（符合无障碍标准）
- 标题：24pt-32pt
- 按钮文字：20pt-24pt
- 支持四种字体大小调节

### 交互规范
- 按钮尺寸：≥48x48dp
- 触摸间距：≥8dp
- 语音反馈：所有操作都有语音确认
- 震动反馈：关键操作提供触觉反馈

## 🔧 技术特性

### 前端技术
- **框架**：React 18 + TypeScript
- **构建工具**：Vite
- **样式**：Tailwind CSS
- **图标**：Lucide React
- **路由**：React Router DOM

### 无障碍技术
- **语音合成**：Web Speech API（粤语）
- **语音识别**：Web Speech API（粤语）
- **震动反馈**：Vibration API
- **字体调节**：CSS自定义属性
- **高对比度**：CSS滤镜和类切换

### 移动端优化
- **响应式设计**：移动端优先
- **PWA支持**：可安装为原生应用
- **触摸手势**：优化的触摸体验
- **离线支持**：基础功能离线可用

## 📋 使用说明

### 对于长者
1. 打开应用即可看到亲切的粤语问候
2. 点击大按钮访问各项功能
3. 使用语音助手时点击麦克风图标
4. 紧急情况下点击红色SOS按钮
5. 通过设置调节字体大小和对比度

### 对于家属
1. 访问 `/family` 查看监控面板
2. 选择要查看的长者用户
3. 查看实时健康数据和趋势
4. 接收紧急警报和提醒
5. 通过快速操作联系长者

## 🛡️ 安全特性

- **紧急求助**：多重确认机制防止误触
- **跌倒检测**：30秒倒计时允许取消
